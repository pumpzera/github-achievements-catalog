# Sources

This repo uses a confidence model on purpose:

- **Official** means GitHub Docs, GitHub Blog, GitHub Changelog, or GitHub-owned program pages.
- **Mixed** means GitHub officially acknowledges the name or concept, but not the full unlock rule.
- **Community** means the row is based on a GitHub Community discussion and should be treated as best-effort guidance.
